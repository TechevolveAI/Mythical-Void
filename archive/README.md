# Archived Documentation

This folder contains:
- **Future feature specifications** that are not yet implemented
- **Historical planning documents** from earlier development phases
- **Business analysis reports** about feature gaps and opportunities

**These documents are kept for reference but do not describe the current game.**

See the root directory README.md for current game features and architecture.

## Folder Structure

### `/planning/`
MVP roadmaps, implementation plans, and phased requirements for future development.

### `/future-features/`
Detailed specifications for features not yet built (e.g., AI chat, LifeCurrent system).

### `/gap-analysis/`
Reports analyzing missing game elements and business opportunities.

### `/reviews/`
Historical review documents from earlier development phases.

### `/vision/`
Long-term vision documents describing the ideal future state of the game.
